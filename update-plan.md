# 这是对于1.0.2版本的更新调整的计划书，你以此为依据，重新修改完善代码。

## 总纲领
更新计划纷繁复杂，你以这个计划书为纲领，以@FIX_PLAN.md 和 @IMPACT_FEATURE_SUMMARY.md两个文件为参考，更新和调试代码。

### 1.更新地图、人物建模、寻路机制

- 当前游戏是1201*676像素的屏幕，能否扩充到全屏，或者至少扩大到你能扩大的极限？不然无法展现细节。

- 当前生成的地图，边界没有封闭，而且没有实心块。要求生成封闭边界，无法进入河流，树木、石头、房屋墙壁是实心块，让人物无法通过。

- 优化房间：生成教堂，里面有实心块设置的讲台、椅子作为障碍物，强制NPC神父生成于教堂，NPC神父限制在教堂内活动；生成居民，里面有床、柜子（每个民宅各一个）等障碍物，居民不被雇佣时在房屋附近活动；所有房屋都有门和墙壁，按交互键'Q'打开；生成墓地，墓地的墙是围栏，里面障碍物是墓碑，生成怪物“不死者”；哥布林随机生成在没有房屋的空旷地带。

- 人物建模： 当前人物只是正方形像素块，做成像素小人，像“Rougue of streets"或者“元气骑士”那种。如果你不能做，搞成大一点的长方形像素块，并告诉我在哪里下载免费可商用素材，或者你自己捏人物出来。

- 寻路机制： 既然有了房屋和障碍物，NPC人物需要使用A*寻路算法，以防止卡死。据此优化寻路机制。

### 2.更新背包武器系统，武器显示，挥击感，射弹和碰撞效果

- 背包：按”E“键可以打开背包，再按”E“键关闭，背包是”5*5“的方格，最多可以储存25个”物品“。设立”物品“类。包括”武器“、”金币“、血瓶、蓝瓶都属于物品大类，可以存储于背包中，掉落在地上，可以按”F“键捡起，放入背包中。设计物品的图标，告诉我你是否可以直接生成，还是需要我导入，如果需要我导入，告诉我详细的操作指南，具体到在哪个网站找。

- 武器系统： 转动”鼠标滚轮“自动切换武器（并在人物头顶冒出白字的武器名字提示），或者左键点击背包里的武器进行更换。武器分为近战武器和远程武器两个大类，远程武器目前只有“圣杖”一个。

- 武器显示：把武器的像素显示在人物身前。

- 挥击感：给近战武器设置”weight“，即重量。重量越大，击退效果越大。“斧头”重量为5，“砍刀”重量为3，“匕首”重量为1，“剑”重量为4，远程武器没有重量。根据重量给击退效果设置权重，并给武器攻击时加上挥动特效（在人物前产生白色弧线）

- 射弹：远程武器释放射弹特效，“神杖”属于远程武器，释放小的黄色像素格子作为射弹。射弹有两种规则消失：一是随时间变化而消失；而是击中角色消失。

- 碰撞效果：被打击而击退后，撞到实心障碍物会以速度的一定几率弹回。

### 3.修复已知bug

- 敌人一碰即死：调高敌对怪物血量，并减少敌对怪物生成数量和攻击力。

- NPC的血条格不会变化：修复该问题，让血条格的红色部分按血量百分比显示，剩余部分用白色填充。

- 商人的技能出了bug：所有角色进入商店都可以打折

- 在商人处买的血瓶和蓝瓶无法使用：设置“H”键，使用血瓶；设置“B”键，使用蓝瓶；也可以打开背包，试用鼠标左键点击背包里对应“物品”使用。
